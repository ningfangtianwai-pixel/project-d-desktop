# Project D 用户帮助

## 开始使用

1. 启动 Project D 后，主界面会显示桌面文件、壁纸、天气和 Luna。
2. 第一次使用建议先打开“设置”，确认桌面整理、壁纸、天气和桌宠选项。
3. Project D 会常驻系统托盘。关闭主窗口不会退出程序；需要完全退出时，请在托盘菜单选择“退出”。

## 整理桌面

- “启动整理”会在真实桌面上显示整理容器，但不会立即移动文件。
- 在执行整理前，请先查看动作预览，确认每个文件的目标位置和冲突处理结果。
- 已完成的整理可在“设置 > 恢复中心”中撤销或回滚。
- “安全归位”会停止桌面接管并恢复 Windows 原生桌面状态。
- “纯净桌面”只隐藏桌面图标，不会删除或移动文件；选择“恢复桌面”即可重新显示。

## 壁纸与天气

- 在“设置 > 壁纸”中选择本地壁纸并应用。
- 动态壁纸可以播放、暂停和循环；全屏应用、省电模式和系统休眠时会自动降低活动。
- 在“设置 > 天气”中可选择自动定位或手动城市。网络暂停时使用最近一次缓存。
- 多显示器布局变化或睡眠唤醒后，Project D 会自动检查并修复壁纸宿主。

## Luna 桌宠与 AI

- Luna 显示在普通窗口和桌面之上，可拖动、复位位置或从右键菜单隐藏。
- 人格和说话频率可在“设置 > 桌宠”中调整。
- AI 对话会携带近期上下文。暂停外部网络后，云端 AI 会自动停用并使用本地提示。
- 请勿在聊天内容中发送密码、支付信息或其他敏感数据。

## 软件更新

- “稳定通道”只接收正式版本。
- “灰度体验”可能提前收到预览版本。灰度发布按匿名设备标识分批开放，不需要账号信息。
- 检测到新版本后，Project D 会先询问是否下载；下载完成后由用户确认重启安装。
- 若显示“尚未配置正式更新服务器”，说明当前测试版尚未连接商业发布站，不影响其他功能。

## 白屏、黑屏或无法关闭

1. 等待数秒。Project D 会自动检测渲染异常并尝试恢复。
2. 若主界面没有恢复，从系统托盘打开“设置 > 恢复中心”，选择“立即恢复”。
3. 若托盘也无响应，可在任务管理器结束 Project D；再次启动时程序会检查数据库、桌面图标和壁纸宿主状态。
4. 仍有问题时，在“设置 > 关于”导出诊断报告。报告会对路径、令牌和密钥进行脱敏。

## 隐私与数据

- 桌面索引、场景、聊天记录和设置默认保存在本机。
- “设置 > 隐私中心”可以暂停所有外部网络请求、导出数据或彻底删除本地数据。
- 卸载程序前请先退出 Project D；卸载不会默认删除用户数据，以便覆盖安装后恢复设置。

## 安全退出与卸载

1. 在托盘菜单选择“安全归位”，确认 Windows 桌面图标正常显示。
2. 在托盘菜单选择“退出”。
3. 再通过 Windows 设置卸载 Project D。

如果异常退出导致桌面图标未恢复，重新启动 Project D 后进入恢复中心执行“立即恢复”。
