# Project D V2 工程说明书

> 产品定位：AI 桌面伴侣（路线 C）
> 核心理念：一个懂你桌面的 AI 伙伴，帮你整理、帮你找、还会陪你
> 编写日期：2026-07-13

---

## 一、产品故事

Luna 不是桌面摆件。她是 AI 的脸。

所有功能通过她来表达：她看到你桌面文件多了会主动问"需要我帮你整理吗？"、她记得你上周改过的文件在哪个文件夹、她会在下雨天换上雨衣然后提醒你"别忘了带伞"。

**Fences 的整理能力 + 360 的搜索效率 + AI 的陪伴感 = Project D。**

### 3.1 拉绳恢复为风格切换

**问题：** 拉绳当前按壁纸名翻页（湖畔车站→海崖晨光→雾岭镜湖），12 张壁纸，翻到后面不知道自己在哪。

**改动：**

文件：`src/renderer/views/OverlayPage.vue`

```typescript
// 改动前：遍历 wallpaperStyleIds（12 个壁纸 ID）
// 改动后：遍历 WALLPAPER_STYLES（6 个风格大类）

const wallpaperStyleCycle = [
  { id: "anime", label: "动漫" },
  { id: "landscape", label: "风景" },
  { id: "cinematic", label: "电影感" },
  { id: "cyberpunk", label: "赛博朋克" },
  { id: "minimalist", label: "极简" },
  { id: "seasonal", label: "季节" }
];

// 每次切换时，从该风格的壁纸中取第一张
async function switchWallpaperStyle(dir: -1 | 1): Promise<void> {
  const idx = wallpaperStyleCycle.findIndex(s => s.id === currentStyle.value);
  const next = wallpaperStyleCycle[(idx + dir + wallpaperStyleCycle.length) % wallpaperStyleCycle.length];
  currentStyle.value = next.id;
  // 取该风格的第一张壁纸
  const firstOfStyle = wallpaperLibrary.value.find(w => w.style === next.id);
  if (firstOfStyle) {
    await window.projectD.applyWallpaper(firstOfStyle.id);
  }
}
```

模板改动：拉绳标签显示 `{{ currentStyleLabel }} · {{ currentWallpaperLabel }}`。

**验收：** 拉绳只切换 6 个风格大类，不逐张翻。标签显示风格名，鼠标悬停显示当前具体壁纸名。

```typescript
// luna-brain.ts
class LunaBrain {
  analyzeDesktop(files: DesktopFileRecord[]): AISuggestion | null {
    // 检测条件 → 生成建议
    // 1. 未分类文件 > 5 → 建议整理
    // 2. 最近 7 天新增 > 10 → 建议归档
    // 3. 重复文件检测（同名/相似名）→ 建议去重
    // 4. 大文件检测（>100MB 的视频/压缩包）→ 建议移动到其他盘
    // 5. 时间：周一早上 → 建议规划本周任务
    // 6. 时间：周五下午 → 建议整理一周文件
  }
}
```

**宠物端触发：**

```typescript
// PetPage.vue
async function checkDesktopCondition(): Promise<void> {
  const files = await window.projectD.getDesktopFiles();
  const suggestion = await window.projectD.getAiSuggestion(files);
  if (suggestion && !bubbleVisible.value) {
    showBubble(suggestion.text, "thinking", 15_000);
  }
}
// 每 15 分钟检查一次
setInterval(checkDesktopCondition, 15 * 60 * 1000);
```

**验收：**
- 桌面文件超过 5 个未分类时，Luna 自动弹出整理建议
- 周一到周五不同时段有不同建议（早上→规划，下午→整理）
- 建议语言符合当前人格设置

### 5.1 AI 桌面搜索（自然语言）

**功能：** 输入"帮我找上周三改过的那份报价单"或"找所有 PDF"，返回匹配的文件列表。

**技术方案：**

```
用户输入 → AI 提取意图 → 生成搜索条件 → 查询本地索引 → 排序返回
```

```typescript
// search-index.ts
class SearchIndex {
  private index: Map<string, FileIndexEntry> = new Map();
  
  buildIndex(): void {
    // 遍历 desktop_files 表，建立内存索引
    // key: fileId, value: { filename, ext, size, modifiedAt, keywords }
  }
  
  searchByAI(intent: SearchIntent): FileIndexEntry[] {
    // intent 由 AI 从自然语言中提取
    // { ext: ".pdf", dateFrom: "2026-07-06", dateTo: "2026-07-13", keyword: "报价" }
    return Array.from(this.index.values()).filter(entry => {
      const matchExt = !intent.ext || entry.ext === intent.ext;
      const matchDate = !intent.dateFrom || new Date(entry.modifiedAt) >= new Date(intent.dateFrom);
      const matchKeyword = !intent.keyword || entry.filename.includes(intent.keyword);
      return matchExt && matchDate && matchKeyword;
    });
  }
}
```

**搜索浮窗 UI：**


## 十、验收标准总表

| 功能 | 验收方法 | 标准 |
|---|---|---|
| 拉绳风格切换 | 手动翻 2 轮 | 6 个风格循环，不逐张翻 |
| 宠物人格短句 | 切换 8 种人格各点一次 | 每种人格有不同风格文案 |
| 工具栏壁纸标签 | 切换壁纸后查看 | 显示当前壁纸名+风格 |
| 聊天输入优化 | 连续发 3 条消息 | 每次发完清空并聚焦 |
| 文件夹门户 | 选 2 个文件夹创建门户 | 内容同步，30 秒刷新 |
| AI 整理建议 | 桌面放 6 个文件等 30 秒 | Luna 弹出建议 |
| 快捷键 | 在浏览器中按 Ctrl+Shift+F | 搜索窗弹出 |
| 待办便签 | 添加 3 条、完成 1 条、删 1 条 | 刷新后数据不丢 |
| AI 桌面搜索 | 输入"找所有 PDF" | 返回匹配文件列表 |
| 桌宠主动建议 | 周一早上打开程序 | 弹出周建议 |
| 容器配色 | 给 3 个容器选不同颜色 | 重启后颜色保留 |
| 布局快照 | 保存→改变布局→恢复 | 位置/门户/配色全部还原 |
| 新手引导 | 删除数据库重新启动 | 3 步引导出现 |
| 数据看板 | 用一周后看统计 | 数据准确 |
